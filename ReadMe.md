# Radio Stations App (React + TypeScript)

project build on Vite
<br/>
<p>Install Dependencies </p>
<code> npm install </code>
<br/>
<p>Run the project </p>
<code> npm run dev </code>
<br/>
<p>Run the Tests </p>
<code> npm run test </code>
<br/>
<p>Please note : use CORS plugin to unblock requests</p>
